# 연대지능 활동가 위키

사회혁신교육원(사교원)의 연대지능활동가 교육 과정에서 축적되는 지식을 함께 만들어가는 공간입니다.

## 이 저장소의 목적

1. **활동가 각자의 학습과 현장 기록**을 체계적으로 축적
2. **검수(리뷰)를 통해 지식의 품질**을 높임
3. **서로의 위키를 연결**하여 집단지성을 형성

## 참여 방법

1. 이 저장소를 **Fork** 합니다
2. 자신의 저장소에서 문서를 작성합니다
3. 정리가 되면 **Pull Request**를 제출합니다
4. 리뷰어의 피드백을 반영하여 **머지**됩니다

자세한 안내는 [온보딩 가이드](docs/onboarding-guide.md)를 참고하세요.

## 폴더 구조

```
wiki/
├── concepts/        # 개념 정리 (AI, 데이터, 협업 도구 등)
├── field-notes/     # 현장 기록 (지역 활동, 주민 만남 등)
├── how-to/          # 실습 가이드 (도구 사용법, 절차 등)
├── case-studies/    # 사례 연구 (성공·실패 사례 분석)
└── resources/       # 참고 자료 링크 모음

docs/
├── onboarding-guide.md   # 활동가 온보딩 가이드
└── review-checklist.md   # 검수 체크리스트

templates/
├── concept.md            # 개념 문서 템플릿
├── field-note.md         # 현장 기록 템플릿
├── how-to.md             # 실습 가이드 템플릿
└── case-study.md         # 사례 연구 템플릿
```

## 검수 기준

모든 기여는 Pull Request를 통해 제출하며, 아래 기준으로 검수됩니다:

| 항목 | 기준 |
|------|------|
| 정확성 | 출처가 있고, 사실과 부합하는가 |
| 완결성 | 맥락 없이도 독립적으로 이해 가능한가 |
| 형식 | 템플릿 구조를 따르는가 |
| 연결성 | 관련 문서와 링크가 적절한가 |
