# 완도 망님마을 인스타그램 콘텐츠 제작 사례

> **한 줄 요약**: AI 도구를 활용해 어촌 마을의 인스타그램 콘텐츠 50개를 체계적으로 기획·제작한 사례

## 사례 개요

완도 망님마을의 어촌신활력증진사업 일환으로, 마을 홍보를 위한 인스타그램 콘텐츠 50개를 AI 도구(Claude)를 활용하여 기획했습니다. 기존에는 콘텐츠 하나를 만드는 데 반나절이 걸렸지만, AI와 협업하여 전체 콘텐츠 기획을 체계적으로 완성할 수 있었습니다.

## 배경과 맥락

- 망님마을은 전남 완도군 금일읍의 어촌 마을
- 어촌신활력증진사업으로 마을 활성화 프로젝트 진행 중
- SNS 홍보가 필요하지만 전담 인력이 없는 상황
- 사교원이 사업 운영을 맡고 있어 콘텐츠 제작도 담당

## 진행 과정

1. 마을의 자원(자연경관, 특산물, 주민 이야기, 행사)을 카테고리별로 정리
2. Claude를 활용해 카테고리별 콘텐츠 아이디어를 브레인스토밍
3. 50개 콘텐츠의 제목, 본문, 해시태그를 JSON 형태로 구조화
4. 계절별, 주제별 발행 순서를 배치
5. 실제 사진/영상 촬영 계획과 매칭

## 결과

- **정량**: 콘텐츠 기획 50개 완성, 기획 소요 시간 약 80% 단축
- **정성**: 콘텐츠 간 일관된 톤앤매너 유지, 계절감 있는 발행 계획 수립, 마을 자원의 체계적 발굴

## 성공/실패 요인 분석

| 요인 | 설명 |
|------|------|
| **잘 된 점** | 마을 자원을 먼저 체계적으로 정리한 후 AI에 입력하여 품질 높은 결과물 도출 |
| **잘 된 점** | JSON 구조화로 나중에 수정·재활용이 쉬움 |
| **아쉬운 점** | 실제 촬영 및 발행까지의 실행 과정은 별도 인력 필요 |
| **외부 요인** | 어촌 마을 특성상 계절별 풍경 변화가 커서 콘텐츠 소재가 풍부했음 |

## 우리에게 주는 시사점

- AI는 "대신 만들어주는 도구"가 아니라 "함께 기획하는 동료"로 활용할 때 효과적
- 현장 자원을 먼저 정리하는 것이 AI 활용의 전제 조건
- 구조화된 출력(JSON 등)을 활용하면 후속 작업의 효율이 크게 향상
- 소규모 마을도 체계적 접근으로 전문적인 콘텐츠 기획이 가능
