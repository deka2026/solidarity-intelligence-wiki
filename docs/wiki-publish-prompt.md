# 위키 배포 프롬프트

활동가가 자신의 작업 결과물을 위키에 올릴 때, 아래 프롬프트를 Claude Code에 붙여넣고 `[  ]` 부분만 채우면 됩니다.

---

## 프롬프트

```
나는 연대지능활동가야. 내 작업 결과물을 연대지능 위키에 올리려고 해.
아래 내용을 온톨로지 원칙에 따라 정리해서 위키 문서로 만들어줘.

## 내 작업 내용
[여기에 자유롭게 작성: 오늘 한 일, 배운 것, 만든 것, 경험한 것 등]

## 기본 정보
- 이름: [활동가 이름]
- 날짜: [활동 날짜 또는 오늘]
- 장소: [해당되면 작성, 없으면 생략]

---

### 온톨로지 적용 규칙 (Claude가 따를 것)

1. **클래스 분류**: 내용의 주된 목적을 판단해서 아래 중 하나로 분류해줘
   - 새로운 개념을 정리한 것 → `wiki/concepts/`
   - 현장 활동을 기록한 것 → `wiki/field-notes/`
   - 도구나 절차의 사용법 → `wiki/how-to/`
   - 사례를 분석한 것 → `wiki/case-studies/`
   - 판단이 애매하면 나에게 물어봐

2. **템플릿 적용**: 해당 유형의 템플릿(`templates/` 폴더)을 따라서 문서를 구성해줘
   - 한 줄 요약, 태그, 작성자, 작성일은 반드시 포함
   - 유형별 고유 속성(장소, 난이도, 소요시간, 지역/대상 등)도 채워줘

3. **용어 통일**: 기존 위키 문서에서 쓰는 용어와 일치시켜줘
   - 위키에 이미 있는 문서의 용어를 확인하고 같은 표현을 써줘
   - 새 용어를 쓸 때는 괄호 안에 풀어써줘

4. **관계 연결**: 기존 위키 문서 중 관련된 것을 찾아서 [[링크]]를 걸어줘
   - 최소 1개 이상의 내부 링크를 포함해줘
   - "관련 문서" 섹션에 정리해줘

5. **중복 확인**: 같은 주제의 문서가 이미 있는지 확인해줘
   - 있으면 새 문서를 만들지 말고, 기존 문서에 추가하는 방향을 제안해줘
   - 없으면 새 문서로 작성해줘

6. **태그 규칙**:
   - 3~5개 작성
   - 기존 문서에서 쓰인 태그를 우선 재사용
   - `#주제` `#지역` `#도구` `#활동유형` 순서로

7. **파일명**: `소문자-하이픈-연결.md` 형식으로 만들어줘 (예: `ai-prompt-basics.md`)

8. **결과물**: 완성된 문서를 내 Fork 저장소의 올바른 폴더에 저장하고,
   셀프 체크 결과를 알려줘:
   - [ ] 템플릿 구조를 따랐는가
   - [ ] 태그 3개 이상인가
   - [ ] 한 줄 요약이 있는가
   - [ ] 관련 문서 링크가 있는가
   - [ ] 파일명이 규칙에 맞는가
   - [ ] 올바른 폴더에 있는가
```

## 사용 예시

### 예시 1: 현장 활동 후

```
나는 연대지능활동가야. 내 작업 결과물을 연대지능 위키에 올리려고 해.
아래 내용을 온톨로지 원칙에 따라 정리해서 위키 문서로 만들어줘.

## 내 작업 내용
오늘 전주 삼천동 마을회관에서 주민 15명과 월례회의에 참석했어.
하반기 마을 텃밭 운영이랑 경로당 프로그램을 논의했는데,
회의록을 손글씨로 쓰고 있어서 디지털화가 필요해 보였어.
회의 끝나고 주민 3명이랑 이야기했는데 "회의 내용을 카톡으로 받으면 좋겠다"고 했어.

## 기본 정보
- 이름: 홍지영
- 날짜: 2026-06-27
- 장소: 전주시 완산구 삼천동 마을회관
```

### 예시 2: 새로운 도구를 배운 후

```
나는 연대지능활동가야. 내 작업 결과물을 연대지능 위키에 올리려고 해.
아래 내용을 온톨로지 원칙에 따라 정리해서 위키 문서로 만들어줘.

## 내 작업 내용
Claude Code를 써서 마을 홍보용 인스타그램 캡션을 만드는 법을 익혔어.
프롬프트에 마을 특산물, 계절, 타겟 연령대를 넣으면 해시태그까지 자동으로 생성해줘.
처음에 "홍보 문구 만들어줘"라고만 했더니 너무 일반적이었는데,
"완도 금일읍 망님마을의 5월 전복 시즌, 30대 여행객 대상"처럼
구체적으로 쓰니까 훨씬 좋은 결과가 나왔어.

## 기본 정보
- 이름: 김민호
- 날짜: 2026-06-27
```

### 예시 3: 프로젝트 사례를 분석한 후

```
나는 연대지능활동가야. 내 작업 결과물을 연대지능 위키에 올리려고 해.
아래 내용을 온톨로지 원칙에 따라 정리해서 위키 문서로 만들어줘.

## 내 작업 내용
강원도 정선 화암면에서 기초생활거점조성사업 입찰을 준비한 과정을 분석했어.
나라장터에서 RFP를 받아서 정량평가표, 정성평가 제안서, 발표자료 3종을
AI 도구로 자동 생성하는 파이프라인을 만들었는데,
수작업으로 1주일 걸리던 걸 하루 만에 완성했어.
다만 HWP 서식 호환에서 폰트 깨짐 문제가 있었고,
표 레이아웃을 수동으로 조정해야 했어.

## 기본 정보
- 이름: 데카
- 날짜: 2026-06-27
- 장소: 강원도 정선군 화암면
```
