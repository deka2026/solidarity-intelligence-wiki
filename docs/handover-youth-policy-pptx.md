# 핸드오버 — 청년정책 제안서 작성 방법론 교육자료: HTML → PPTX 4종 변환

> 작업 폴더: `C:\Users\pc\클로드로컬\`

## 한 줄 요약

2026 인천 청년정책 공모전 본선 20개 팀 분석 기반 교육자료(HTML 슬라이드 8장, 다크 테마, 버전1/2 토글 내장)를 **pptxgenjs(Node)로 PPTX 4종**(다크/라이트 × 전체 8장/팀분석 제외 7장)으로 변환하고, 한쇼(Hancom Show) "손상된 문서" 경고를 해결했다.

|------|------|
| `청년정책_제안서_작성_방법론.pptx` | 다크 테마, 전체 8장 (팀 분석 표 + 부록 포함) |
| `청년정책_제안서_작성_방법론_버전2.pptx` | 다크 테마, 7장 (팀 분석 표·부록 제외, 프로세스 3단 카드 재배치) |

## 진행 경과

1. **1차 변환**: 16:9 와이드, 맑은 고딕, HTML 다크 테마 재현. PowerPoint 기준으로는 정상.
2. **한쇼 손상 경고**: "일부 텍스트, 이미지, 개체 등이 손상된 문서" — pptxgenjs가 만드는 빈 노트 슬라이드/노트 마스터의 스키마 순서 위반이 원인. jszip 후처리로 노트 부품 전체 제거해 해결 → [[skill-pptxgenjs-hancom-compat]]
3. **가독성 개편(사용자 피드백 2회)**: 여백 축소(좌우 0.6→0.4"), 글자 확대(본문 11→13.5~15pt, 표 10.5→13pt), 본문색 거의 흰색(F1F5F9), 카드·강조박스 대비 강화, 중간 공백 제거(박스를 위로 당기고 높이 확대).
4. **라이트 테마 신규 2종**: 팔레트만 반전(흰 배경 + 다크 네이비 텍스트 + 연한 색 강조 박스), 레이아웃 동일.

## 주의 (다음 작업자용)

- 사용자가 한쇼로 파일을 열어둔 채 재생성하면 **EBUSY 잠금으로 덮어쓰기 실패** — 스크립트가 temp 생성 후 copy 시도, 실패 시 `_새` 접미사로 저장하게 되어 있음. 어떤 파일이 최신인지 사용자에게 반드시 명시할 것.
- 이 PC에는 python·LibreOffice가 없어 렌더링 검수(이미지 변환) 불가 — 육안 검수는 사용자 몫. 구조 검증은 PowerShell(ZipFile + XmlDocument)로만 수행.
- 슬라이드 내용 수정은 스크립트의 텍스트 상수만 고치고 재실행하면 됨. 4종 모두 갱신하려면 두 스크립트 다 실행.
