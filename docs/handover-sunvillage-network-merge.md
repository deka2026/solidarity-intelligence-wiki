## 중복 제거 판단

- **수익 계산기**: 업무자동화 것이 더 정교 → 유지, 지역목록만 전남·광주 22개 시군으로 확대.
- **소개**: 네트워크 홈 콘텐츠(참여자격·6단계·지원·의무)를 무료 소개 탭으로 편입.
- **질문답변(사람 답변)** vs **AI 상담(즉시 답변)**: 성격이 달라 **둘 다 유지**(중복 아님).
- 공고문·서식·일정 등 업무자동화 기존 기능은 그대로 보존.
- **서류관리(체크리스트 트래커)** 는 AI 문서작성/검토와 별개 → 신규 통합 탭.

## 구현 방식 (중요)

- **베이스 = sunvillage.html을 그대로 복사**한 뒤 위에 레이어링. 2,475줄 검증된 AI 로직을 재작성하지 않고 보존하기 위함.
- 적용한 편집:
  1. 리브랜드(title·logo)
  2. 헤더 우측 "마을 칩" → **로그인/회원 배지(authbox)** 로 교체
  3. 탭바를 `data-sec`/`data-need` 속성 기반 3계층으로 재구성 (기존 위치 기반 index map은 탭 추가 시 깨지므로 폐기)
  4. `go()` 함수를 게이팅+새 렌더러 디스패치로 재작성
  5. 다크 테마용 신규 컴포넌트 CSS 블록 추가(table.dt, hero, feat, doc-row, qa-item, gs-check, rbadge 등)
  6. 신규 섹션 HTML(소개·가입문의·서류관리·경영공시·조합KPI·질문답변·관리자·auth 모달) 주입
  7. 네트워크 JS 블록(인증·문의·QA·서류·공시·KPI·관리자) 추가 + init 갱신
  8. AI 오류 메시지를 우아하게(`aiErrMsg()`) — 서버 미연결 시 안내 문구
